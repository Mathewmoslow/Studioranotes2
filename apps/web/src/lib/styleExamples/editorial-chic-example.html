FOUNDATION

Document Formatting Standards

This guide demonstrates the professional formatting style used in high-quality study materials and educational documents across all academic disciplines. The formatting emphasizes clarity, elegance, and visual hierarchy to enhance learning and retention through sophisticated, chic design principles.

CORE PHILOSOPHY

Clean, Crisp, and Professional: All formatting choices prioritize readability and sophistication. No rounded corners, no excessive decoration—just elegant, timeless design that puts content first while maintaining visual interest through strategic use of typography, color, and white space.

TYPOGRAPHY

Header Hierarchy

Primary Headers (H2)

Large serif headers with a top border and accent line. These mark major section breaks and create visual rhythm throughout the document.

PRIMARY SECTION EXAMPLE

Major Topic Area

Primary headers use Libre Baskerville at 2.5rem (40px) with a 3px black top border. A colored accent line (80px wide) overlays the left portion of the border, creating an elegant detail that reinforces your brand colors.

  • Font: Libre Baskerville, 700 weight
  • Top border: 3px solid black
  • Accent: 80px colored line overlay
  • Spacing: 4rem top, 2rem bottom

Secondary Headers (H3)

Smaller serif headers in accent color for subsections within major topics.

CONTENT BLOCKS

Information Box Types

Different box types communicate different information hierarchies. Each uses a bold 4px left border and subtle background tint—no rounded corners, just clean lines.

Definition Box (Blue)

STANDARD INFORMATION

Primary information box for definitions, key concepts, and foundational knowledge.

  • Border: 4px solid left border in accent blue (#1e3a8a)
  • Background: Very light blue tint (#eff6ff)
  • Shadow: Minimal (0 2px 4px rgba(0,0,0,0.06))
  • Use case: Definitions, standard information, key concepts

Information Box (Brown/Warm)

SECONDARY INFORMATION

Secondary information box for related details, context, or supporting content.

  • Border: 4px solid left border in warm brown (#7c2d12)
  • Background: Very light warm tint (#fef9f5)
  • Shadow: Minimal (0 2px 4px rgba(0,0,0,0.06))
  • Use case: Related information, examples, context

Callout Box (Black Border)

CRITICAL CONCEPT

High-emphasis callout with full black border for the most important information.

  • Border: 2px solid black on all sides
  • Background: Pure white (#ffffff)
  • Shadow: Minimal (0 2px 4px rgba(0,0,0,0.06))
  • Use case: Critical concepts, essential information, must-know content

Remember Box (Teal/Green)

Subtle reminder box for key takeaways, memory aids, or important notes. Uses a teal/green accent for "remember this" content without being visually overwhelming.

Warning / Critical Alert Box (Red)

CRITICAL ALERT

Critical alerts demand attention. This box uses deep red borders (6px on left, 1px top/bottom) and red-tinted background to immediately signal important warnings, common errors, or information requiring urgent attention. Strong visual hierarchy ensures these cannot be missed.

  • Border: 6px left + 1px top/bottom in deep red (#991b1b)
  • Background: Light red tint (#fef2f2)
  • Shadow: More prominent (0 2px 8px) for extra emphasis
  • Use case: Common mistakes, critical warnings, pitfalls to avoid

DESIGN SYSTEM

Typography Guidelines

FONT FAMILIES

Two Typefaces, Maximum Impact

  • Libre Baskerville (Serif): Headers and display text—elegant, authoritative, timeless
  • Work Sans (Sans-serif): Body text and labels—clean, highly readable, modern

This pairing creates sophisticated contrast between authoritative headers and approachable body content.

SUMMARY

Key Design Principles

Clean Lines

No rounded corners. Sharp, crisp edges create a sophisticated, professional appearance that ages well.

Restrained Color

Deep jewel tones used sparingly. Let typography and white space do the heavy lifting.

Elegant Typography

Serif headers paired with clean sans-serif body. Two fonts maximum for cohesion.

Bold Borders

4px left borders create emphasis without overwhelming. Strategic use draws the eye.

Minimal Shadows

Subtle elevation (0 2px 4px) provides depth without gimmicks or excess decoration.

Generous Spacing

White space is a design element. Don't fear emptiness—it creates breathing room and focus.

When to Use Each Box Type

Information Hierarchy

  • Definition box (blue): Primary concepts, definitions, key information
  • Info box (brown): Secondary details, context, related information
  • Callout box (black border): Critical must-know information
  • Remember box (teal): Key takeaways, memory aids
  • Warning box (red): Common mistakes, critical warnings, pitfalls to avoid

Design Consistency

  • Use the same box types: Throughout your document for similar content
  • Maintain color associations: Once blue = definitions, keep it that way
  • Red means caution: Reserve red warning boxes for true warnings only
  • Limit box variety: Don't use all five types on one page
  • Balance emphasis: If everything is highlighted, nothing stands out
FINAL THOUGHTS

Sophistication Through Restraint: Chic design isn't about adding more—it's about thoughtful curation. Every element serves a purpose. Every color choice is intentional. The result is timeless elegance that puts your content front and center while maintaining visual interest through:

  • Strategic use of typography hierarchy
  • Restrained, sophisticated color palette
  • Clean, crisp lines with no unnecessary decoration
  • Generous white space that gives content room to breathe
  • Consistent patterns that create visual rhythm
  • Professional details like subtle shadows and border accents
Remember: The goal is elegant professionalism—documents that communicate authority and care through design. When in doubt, simplify. When tempted to add decoration, step back. Less is more, and quality always trumps quantity. Your content deserves presentation that matches its substance.
